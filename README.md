# Bristlecone
Documenting utility to track CTF/Red Team engagements in the terminal.

![ui](resources/ui.png)

## Installation
```bash
git clone https://github.com/Apis-Carnica/Bristlecone.git
cd Bristlecone
pip install -r requirements.txt
```

## Usage
```bash
python bristlecone.py
```
